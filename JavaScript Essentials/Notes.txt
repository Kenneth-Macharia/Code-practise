JavasScripts is a scripting language that allows the craftign of small programs

Atom-live-server:An atom package for updating the browser automatically when a
file is changed. It aslso simulates the written JavaScript, as if it's being served
from a real server. It achieves this by spinnig up a small web server in atom.
-Use Ctr+Alt+l to launch the server on the project folder. Navigate to a folder
